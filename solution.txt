And a written explanation of your solution

The solution provides every possible substring from a given string by initially creating an empty array called substrings.
The outer loop determines where each substring begins. The inner loop chooses where the substring ends. It then takes the two positions
and gives every possible consecutive substring and pushes those results to the empty array. After that it returns the updated array
with all substrings present. 

Big O Time Complexity

The time complexity of this solution is O(n²) due to the use nested loops to examine every possible starting and ending position 
in the string.
